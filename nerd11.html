Nerd 11: Pre CSS day

Pre CSS day

Samenvatting

Tijdens de Pre CSS Day kregen we eerst een korte samenvatting van hoe deze minor is opgebouwd, inclusief de vakken die we volgen en voorbeelden van projecten die zijn gemaakt. Sanne presenteerde zijn werk en toonde projecten van de cursus "CSS to the Rescue" en de hackathon. Daarna nam Bramus het over. Bramus, een voormalig leraar en nu Chrome-ontwikkelaar bij Google, besprak SPA (Single Page Application) view transitions. Hij legde uit hoe de nieuwe View Transition API in Chrome naadloze visuele overgangen tussen verschillende weergaven mogelijk maakt. Hij demonstreerde dit met voorbeelden, zoals de live-transitions van Maxi Ferreira. Bramus liet zien hoe je een view transition start met document.startViewTransition() en hoe je deze transitions kunt aanpassen met CSS, zoals het instellen van de duur en het gebruik van keyframes. Hij vermeldde ook dat cross-document view transitions momenteel alleen in Chrome werken en bepaalde voorwaarden vereisen, zoals dezelfde URL-origin en een opt-in via @view-transition { navigation: auto; }. Het tweede deel van de lezing werd verzorgd door Miriam Suzanne, die sprak over container queries. Helaas heb ik dit gedeelte gemist.

Reflectie

Hoewel de inhoud van de lezing interessant was, vond ik het tempo vrij hoog, waardoor het soms moeilijk te volgen was. De informatie van Bramus over de View Transition API was leerzaam, maar ook complex. De locatie en inrichting van de sessie waren leuk en gaven een andere dynamiek, maar ik merk dat ik de kleinere, meer intieme Weekly Nerd sessies toch fijner vind. Al met al gaf deze sessie me wel nieuwe inzichten, maar door het snelle tempo heb ik niet alles kunnen absorberen. Daarom geef ik deze Weekly Nerd een beoordeling van 6/10.
